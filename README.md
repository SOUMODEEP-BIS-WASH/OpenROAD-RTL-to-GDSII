# OpenROAD-RTL-to-GDSII

## Overview
This repository contains ASIC implementation projects completed using the OpenROAD flow.

The projects demonstrate complete digital backend implementation from RTL design to final GDSII generation.

## Flow Used
- RTL Design
- Logic Synthesis
- Floorplanning
- Placement
- Clock Tree Synthesis (CTS)
- Routing
- Timing Analysis
- Power Analysis
- Final GDSII Generation

---

## Projects Included

### 1. LFSR
- Pseudo-random sequence generator
- Implemented using Verilog
- Completed full RTL-to-GDSII flow using OpenROAD, Yosys, IcarusVerilog, GTKWave and Klayout

### 2. Bidirectional Shift Register
- Supports left and right shifting
- Implemented using Verilog
- Completed full RTL-to-GDSII flow using OpenROAD, Yosys, IcarusVerilog, GTKWave and Klayout
